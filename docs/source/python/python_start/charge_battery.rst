3. 树莓派电源供给（重要）
=====================================================

充电
-------------------

先插入电池线，再接入 USB-C 数据线即可为电池充电。  
需要自备充电器，推荐使用 5V 3A 充电器，普通智能手机充电器也可满足需求。

.. image:: img/BTR_IMG_1096.png

.. note::
    将外接 Type-C 电源接入 Robot HAT 的 Type-C 接口后，会立即开始为电池充电，同时红色指示灯会亮起。  
    当电池充满时，红灯会自动熄灭。


开机
----------------------

拨动电源开关，电源指示灯及电池电量指示灯将同时点亮。

.. image:: img/BTR_IMG_1097.png

等待数秒后，你会听到轻微的蜂鸣声，这表示树莓派已成功启动。

.. note::
    如果电池电量指示灯全部熄灭，请及时为电池充电。  
    在长时间编程或调试过程中，可以保持 USB-C 数据线接入，为电池充电的同时维持树莓派正常运行。
