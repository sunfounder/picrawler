姿勢の調整
==========================

この例では、リモート機能を使ってPiCrawlerの足を1本ずつ制御し、所定の姿勢を取らせます。

ボタンをタップすることで、現在の座標値を表示できます。これらの座標値は、PiCrawlerのユニークなアクションを作成する際に役立ちます。
