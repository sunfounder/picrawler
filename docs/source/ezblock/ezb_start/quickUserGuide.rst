快速用户指南
=====================================

这里是使用EzBlock的一些使用指南。在组装的时候，你只需要完成 :ref:`install_ezblock_os_latest` 和 :ref:`servo_adjust`。

组装完成后再去 :ref:`download_ezb`, 完成 :ref:`connect_product_ezblock_latest` 后，就可以根据教程内容对你的机器人进行编程了。

建议按顺序阅读课程内容。

EzBlock 3.2 的主要优化是将在线服务切换为离线模式，并新增项目文件的导入和导出功能。
届时，新用户注册通道将关闭，已注册用户登录后会收到提示，要求将项目转移至离线模式。
在线服务将在一段时间后关闭。

.. note::
    3.2 更新无需为树莓派重新烧录镜像。
