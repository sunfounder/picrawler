Robot HAT
==================

|link_robot_hat_v4| 是一款多功能扩展板，可让 Raspberry Pi 快速变身为机器人。  
板载 MCU 用于扩展 Raspberry Pi 的 PWM 输出与 ADC 输入，  
同时集成电机驱动芯片、I2S 音频模块及单声道扬声器，  
并保留了 Raspberry Pi 自身的 GPIO 接口。

扩展板还配备了一个扬声器，  
可用于播放背景音乐、音效，甚至实现 TTS 功能，让你的项目更具趣味性。

支持 7-12V 电源输入，配有 2 个电池指示灯、1 个充电指示灯和 1 个电源指示灯。  
此外，还提供一个用户可用的 LED 与一个按键，方便你快速测试功能效果。

更多详细说明，请参考：|link_robot_hat_v4|。
