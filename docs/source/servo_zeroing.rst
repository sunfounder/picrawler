舵机归零以便组装
=================================

在组装舵机之前，  
需要先将角度设置为零位。  
这是因为舵机电机的运动范围有限，  
将角度设为零度可以确保舵机处于初始位置，  
在通电时不会超出运动范围。  
如果在组装前未将舵机归零，  
通电后舵机可能会尝试超出其运动范围，  
从而导致舵机或所连接的机械系统损坏。  
因此，将舵机设为零位是确保其安全、正常运行的重要步骤。
